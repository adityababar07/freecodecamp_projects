Input/Output

In Python, input is the process of getting data from the user. This data can be anything from a simple string to a complex object. The most common way to get input from the user is to use the input() function. The input() function takes a prompt as its argument, which is a string that is displayed to the user before they enter their input. The user's input is then returned as a string. For example, the following code will ask the user for their name and then print it back to them: name = input("What is your name? ") print("Your name is", name)

Output in Python is the process of displaying data to the user. This data can be anything from a simple string to a complex object. The most common way to output data to the user is to use the print() function. The print() function takes any number of arguments, which are the data that you want to display. The print() function will then display the data to the user, one item per line. For example, the following code will print the numbers 1 to 10: for i in range(1, 11): print(i)

Operators

1 Operators in Python are special symbols that are used to perform operations on variables and values. There are many different types of operators in Python, but some of the most common include: Arithmetic operators are used to perform mathematical operations, such as addition, subtraction, multiplication, division, and modulus. For example, the expression 5 + 2 will return the value 7, because 5 added to 2 is 7. Comparison operators are used to compare two values and return a Boolean value (True or False). For example, the expression 5 == 2 will return False, because 5 is not equal to 2. Logical operators are used to combine Boolean values and return a new Boolean value. For example, the expression 5 > 2 and 3 < 4 will return True, because 5 is greater than 2 and 3 is less than 4. Assignment operators are used to assign values to variables. For example, the expression x=5 will assign the value 5 to the variable x. Identity operators are used to compare the identity of two objects. For example, the expression x is y will return True if x and y are the same object, and False otherwise. Membership operators are used to check if a value is a member of a sequence. For example, the expression 'a' in 'abc' will return True, because the character 'a' is a member of the string 'abc' .

Here is a table of all the operators in Python: Operator Description Arithmetic operators +, -, *, /, //, % Comparison operators ==, !=, <,>, <=,>= Logical operators and, or, not Assignment operators =, +=, -=, *=, /=, //=, %= Identity operators is, is not Membership operators in, not in For more information on operators in Python, you can refer to the Python documentation: https://docs.python.org/3/library/operator.html.

Datatypes

Data types in Python are used to define the type of data that a variable can store. There are many different data types in Python, but some of the most common include: Numeric data types are used to store numbers. The most common numeric data types are int, float, and complex. String data types are used to store text. Strings are enclosed in quotes. Sequence data types are used to store collections of data. The most common sequence data types are list, tuple, and range. Mapping data types are used to store data in key-value pairs. The most common mapping data type is dict. Boolean data types are used to store true or false values. The only two possible values for a Boolean data type are True and False.

Here is a table of all the data types in Python: Data type Description int Used to store integers float Used to store floating-point numbers complex Used to store complex numbers str Used to store strings list Used to store lists of data tuple Used to store tuples of data range Used to store a range of numbers dict Used to store mappings of keys to values bool Used to store Boolean values For more information on data types in Python, you can refer to the Python documentation: https://docs.python.org/3/library/stdtypes.html. Control Flow

Control flow in Python is the order in which the statements in a program are executed. It is regulated by conditional statements, loops, and function calls. Conditional statements are used to make decisions about the flow of execution. The most common conditional statement in Python is the if statement. The if statement evaluates a condition and, if the condition is true, executes a block of code. If the condition is false, the block of code is skipped. x = 5 if x < 10: print("x is less than 10") else: print("x is not less than 10") Learn more Loops are used to repeat a block of code a certain number of times or until a certain condition is met. The most common loops in Python are the while loop and the for loop. The while loop repeats a block of code as long as a condition is true. The for loop repeats a block of code for each item in a sequence. # While loop x=0 while x < 10: print(x) x +=1 # For loop numbers=[1, 2, 3, 4, 5] for number in numbers: print(number) Learn more Function calls are used to execute a block of code that has been defined previously. Functions can be used to encapsulate code that is used frequently or to make code more modular. def factorial(n): if n==0: return 1 else: return n * factorial(n - 1) print(factorial(5))

These are just a few examples of the control flow statements in Python. There are many other control flow statements available, and you can learn more by exploring the Python documentation.
